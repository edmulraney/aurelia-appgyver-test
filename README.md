# aurelia-appgyver-test

Clone this package into the root of a Steroids application to turn it into an Aurelia Steroids app.

e.g.

```steroids create myapp```

```cd myapp```

```git clone https://github.com/edmulraney/apprelia.git```

```cd apprelia```

```gulp build```

```cd ../app```

```gulp watch```
